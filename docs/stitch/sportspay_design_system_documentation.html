# SportsPay Design System (Arena Velocity) ## 1. Visual Foundation ### Color Palette * **Primary:** #1B5E20 (Deep Green) - Used for primary buttons, FABs, and active navigation states. * **Secondary:** #AEEA00 (Vibrant Lime) - Used for energy accents and specific status indicators. * **Background (Light):** #F8F9FA (Surface), #FFFFFF (Cards). * **Background (Dark):** #121212 (Surface), #1E1E1E (Cards). * **Typography:** #212121 (Headings), #616161 (Body). * **Status Colors:** * Success: #43A047 (Confirmed) * Warning: #FB8C00 (Pending) * Error: #E53935 (Declined/Danger) ### Typography * **Headings:** Plus Jakarta Sans, Bold. Used for screen titles and card headers. * **Body:** Inter, Regular/Medium. Used for labels, details, and secondary information. ### Containers & Elevation * **Cards:** 16px corner radius, 16px internal padding, subtle 2px shadow. * **Buttons:** 12px corner radius for standard buttons, 24px for specific bottom sheets or large actions. * **Floating Action Button (FAB):** 56px circle, positioned 24px from screen edges. ## 2. Shared Components ### TopAppBar * **Styles:** * `small`: Centered or left-aligned title, leading back icon, optional trailing action/icon. * **Theming:** Light background (#F5F6F7) or Dark (#121212). ### BottomNavBar * **Styles:** * `label_icon`: Standard navigation with icons and labels. * **States:** Active item uses #1B5E20 (Light) or #B9F61D (Dark). ### Cards & Lists * **Group Card:** Sport icon (circle), Group Name (Bold), Participant Count, Next Event. * **Event Card:** Event Name, Date/Time/Location, Status Badges (Pills). * **Participant Row:** Avatar, Name, Status Toggle Buttons (✓, ?, ✗). ### Forms & Inputs * **Text Input:** Rounded with subtle borders, clear labels. * **Radio Buttons:** Large 2x2 grid for sport selection with active state (green border/tint). * **Mode Selector:** Pill-style toggle buttons for switching views (e.g., "Por times", "Manual"). ## 3. Specialized Layouts * **Team Result Cards:** Colored headers (Green for A, Blue for B, Orange for C) with vertical player lists. * **Instagram Story Template:** 9:16 vertical layout, high-impact background, carousel-style team cards. * **Danger Zone:** Light red tint background, bold red text for destructive actions.
